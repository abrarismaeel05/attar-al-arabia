Project Overview

This project is a 3D fragrance gallery called Attar Al-Arabia, built as the final assignment for H7006 Web 3D Applications at the University of Sussex. I wanted to pick a theme that was original and something I was actually interested in, so I based it around Arabian perfumery.

The word "attar" comes from Arabic and means fragrance or essential oil. It is one of the oldest perfumery traditions in the world and it felt like a strong concept to build a 3D gallery around. The site presents three original perfume bottles that users can interact with in the browser using Three.js and WebGL.

Technology Used

The whole application runs in the browser using standard open source web technologies. No special software is needed to view it.

  • Three.js (r160) is the core library for all 3D rendering. It handles the scene, camera, lights, geometry and the animation loop, sitting on top of WebGL so I did not need to write raw WebGL code.
  • GLTFLoader loads the 3D bottle models exported from Blender as GLB files, which is a compact binary format that packages textures and animations together.
  • OrbitControls is a Three.js addon that handles the mouse controls for rotating, panning and zooming around the model.
  • dat.GUI is a small library I used to build the interactive lighting panel on the right hand side of the viewer.
  • Bootstrap 5 provides the responsive grid layout and the navigation bar.
  • Blender is the 3D modelling software I used to create the bottle shapes, set up materials and export to GLB format.
  • HTML5 and CSS3 for structure and styling. All custom CSS was written by hand.
  • ES Modules with Import Maps is the modern way to import Three.js without needing a build tool like Webpack, which kept the setup much simpler.

3D Modelling Process

All three bottles were modelled in Blender. I wanted each bottle to have a shape that matched the character of the fragrance, so I thought about the silhouette of each one before starting to model.

  • Planning the shape. Oud Noir is tall and angular to feel bold and dark. Rose Royale is round and bulbous to feel soft. Amber Essence is faceted like a cut gem to feel warm and premium.
  • Building the geometry. I used the Lathe modifier on a curve profile in Blender to get a smooth organic bottle body. The neck, stopper and cap are separate cylinder objects attached to the body.
  • Setting up materials. All materials use the Principled BSDF shader. The glass bodies have some transmission to look slightly see-through. The gold caps use a high metalness value.
  • Adding animation. Each bottle has a simple rotation keyframe animation so there is something to play when the Animate button is clicked in the app.
  • Exporting. I exported each model using File, Export, glTF 2.0 and selected the GLB binary format with Animation enabled. Each file saved into the assets/models folder.

Testing

Testing was done during Week 11 using three methods from the Lab 11 content.

  • Functional testing. I went through every button and feature one by one to check they all worked correctly. This included the model selectors, Animate, Wireframe, Rotate, Reset, the dat.GUI sliders and the dropdown navigation.
  • Cross-browser testing. I tested in Chrome, Firefox and Safari to check the Three.js scene and ES module import maps worked in each browser.
  • Responsive testing. I used Chrome DevTools device emulation to check the layout at mobile (375px) and tablet (768px) screen widths.
  • Peer usability testing. During the Week 11 lab I asked another student to use the site without any instructions from me. They were able to load a model, toggle wireframe and adjust the lighting without needing any help.
  • Console debugging. I kept Chrome DevTools open throughout development to catch errors early. The final build has no unhandled errors in the console.
